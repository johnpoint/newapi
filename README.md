# newapi

一个用于快速生成 Go API 样板代码的 CLI 工具，基于 [gin-gonic](https://github.com/gin-gonic/gin) 和 [go-bootstrap](https://github.com/johnpoint/go-bootstrap) 框架。

## 功能

- **`new`** 命令：通过命令行参数快速创建单个 API
- **`gen`** 命令：通过 YAML 配置文件批量生成多个 API

每个 API 会自动生成以下三个文件：

| 文件 | 说明 |
|------|------|
| `endpoints/{name}_ep.go` | Endpoint 层，绑定路由与 HTTP 方法 |
| `services/{name}_srv.go` | Service 层，包含业务逻辑与 Swagger 注解 |
| `schemas/{name}_schema.go` | Schema 层，定义请求/响应结构体 |

> 注意：若文件首行包含 `Code generated by newapi. DO NOT EDIT.`，则重新执行时会覆盖该文件；否则保留已有内容（schema 文件从不覆盖）。

## 安装

```bash
go install newapi@latest
```

或者克隆后本地构建：

```bash
git clone https://github.com/johnpoint/newapi.git
cd newapi
go build -o newapi .
```

## 使用 Docker

### 构建镜像

```bash
docker build -t newapi .
```

### 运行

> 将宿主机的 Go 项目目录挂载到容器内的 `/workspace`，项目根目录须包含 `go.mod`。

**`new` 命令 — 生成单个 API：**

```bash
docker run --rm -v $(pwd):/workspace newapi new \
  -n GetUser \
  -d "获取用户信息" \
  -t user \
  -m get \
  -p /user/{id}
```

**`gen` 命令 — 从 YAML 批量生成 API：**

```bash
docker run --rm -v $(pwd):/workspace newapi gen -c api.yaml
```

> Windows PowerShell 用户请将 `$(pwd)` 替换为 `${PWD}`：
>
> ```powershell
> docker run --rm -v ${PWD}:/workspace newapi gen -c api.yaml
> ```

## 使用

### `new` — 创建单个 API

```bash
newapi new [flags]
```

| Flag | 简写 | 默认值 | 说明 |
|------|------|--------|------|
| `--name` | `-n` | `TestApi` | API 名称（大驼峰命名） |
| `--desc` | `-d` | `auto generate api` | API 描述 |
| `--tag` | `-t` | `test` | Swagger Tag |
| `--method` | `-m` | `get` | HTTP 请求方法 |
| `--path` | `-p` | `/ping` | 请求路径（支持路径参数，如 `/user/{id}`） |

**示例：**

```bash
newapi new -n GetUser -d "获取用户信息" -t user -m get -p /user/{id}
```

生成文件：
- `endpoints/get_user_ep.go`
- `services/get_user_srv.go`
- `schemas/get_user_schema.go`

### `gen` — 批量生成 API

```bash
newapi gen -c <配置文件.yaml>
```

**配置文件格式（YAML）：**

```yaml
group:
  - name: user
    apis:
      - name: GetUser
        path: /user/{id}
        desc: 获取用户信息
        method: get
      - name: CreateUser
        path: /user
        desc: 创建用户
        method: post
  - name: order
    apis:
      - name: GetOrder
        path: /order/{id}
        desc: 获取订单
        method: get
```

**示例：**

```bash
newapi gen -c api.yaml
```
